Donner Software Tools

This is the primary double elimination competition tool. This will mimic key aspects of a double elimination; competitors are ejected after 2 loss, winners only compete against winners until the finals; and also implement some key advantages; members on the same team won't be paired together if there is another suitable match, competitors can be added and removed throughout the competition (not recommended if stakes are high). It is also incredibly simple to use, and allows for events to be saved and recalled!

This is a variable elimination competition tool. Ideal for single elimination, or round robin-esque matches where you don't want to eliminate a competitor until they have lost X times. Use the optimized tool (left) for double elimination competitions.
